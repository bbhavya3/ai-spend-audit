# AI Spend Audit

AI Spend Audit is a web application that helps startups identify overspending on AI tools like ChatGPT, Claude, Cursor, and GitHub Copilot.

Users can calculate potential savings, receive optimization recommendations, and understand where their AI budget can be reduced.

## Features

- AI tool spend calculator
- Monthly and annual savings estimation
- Dynamic audit recommendations
- Persistent form data using localStorage
- Responsive modern UI
- Startup-focused landing page

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Run Locally

```bash
npm install
npm run dev
```
