# Reflection

## 1. Hardest Bug

The hardest issue during development was handling persistent form data using localStorage in React components. Initially, the form values reset every time the page refreshed. I researched how React useEffect works and implemented localStorage save and retrieval logic. After debugging state updates and render timing, the persistence started working correctly.

## 2. Decision I Reversed

Initially, I planned to create a very large and complex audit engine with multiple pages and advanced calculations. Midway through development, I realized a simpler and cleaner MVP would provide a better user experience and allow faster iteration. I reduced unnecessary complexity and focused on core functionality first.
